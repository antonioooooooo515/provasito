Html

L’HTML è un linguaggio per la scrittura di documenti ipertestuali che è stato studiato per essere utilizzato all’interno del WWW (World Wide Web). È basato sia sul protocollo per la trasmissione di documenti (HTTP), che sulla notazione per l’indirizzamento delle risorse (URL). Le pagine di un sito (costituite da documenti HTML) sono memorizzate su un server Web e vengono visualizzate dal computer client attraverso l’utilizzo di un browser Web, come per esempio Chrome, Firefox, Opera, Safari e Internet Explorer. Quando l’utente digita l’indirizzo Web (URL) della pagina che vuole visitare, il sistema operativo richiede, attraverso una connessione HTTP, la pagina al server in cui è memorizzata e la scarica affinché possa essere visualizzata dal browser. Quando il server non trova la pagina, per errore di digitazione oppure per altri motivi, restituisce il codice di errore 404.

Css

Le pagine Web scritte in HTML non consentono di definire un layout adeguato, in quanto i vari elementi di cui è formato un documento (intestazione, barra di navigazione, piè pagina ecc.) possono essere collocati soltanto attraverso le tabelle. Grazie ai fogli di stile, invece, possiamo costruire il contenuto e successivamente applicarvi lo stile desiderato semplicemente variando la pagina dello stile.
