Html

L'HTML (HyperText Markup Language) è l'elemento costitutivo più basilare del Web. Definisce il significato e la struttura dei contenuti web. Altre tecnologie oltre all'HTML vengono generalmente utilizzate per descrivere l'aspetto/presentazione (CSS) o la funzionalità/comportamento di una pagina Web (JavaScript). "Ipertesto" si riferisce a collegamenti che collegano le pagine Web tra loro, all'interno di un singolo sito Web o tra siti Web. I link sono un aspetto fondamentale del Web. Caricando contenuti su Internet e collegandoli a pagine create da altre persone, diventi un partecipante attivo al World Wide Web. HTML utilizza il "markup" per annotare testo, immagini e altro contenuto da visualizzare in un browser Web. Il markup HTML include "elementi" come per esempio "p", per la creazione di paragrafi. Un elemento HTML è separato da altro testo in un documento da "tag", che consistono nel nome dell'elemento circondato da "<" e ">". Il nome di un elemento all'interno di un tag non fa distinzione tra maiuscole e minuscole. Cioè, può essere scritto in maiuscolo, minuscolo o misto. I tre tag principali sono <html>, <head>, utilizzato per l'intestazione del file e <body>, utilizzato per il corpo del file.

Css

Chiamati anche CSS (Cascading Style Sheets), sono un insieme di comandi di formattazione di documenti HTML, XHTML se XML utilizzabili nelle pagine Web. Consentono di separare icontenuti dalla formattazione e permettonoanche di formulare una programmazione più chiara e leggibile, sia per gli autori delle pagine HTML che per gli utenti, garantendo contemporaneamente anche il riutilizzo del codice e una sua più facile manutenibilità.
