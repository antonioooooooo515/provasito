Layout


Relativamente alla disposizione dei contenuti, possiamo distinguere i layout in base al numero di colonne utilizzate.


I layout più comuni sono:

  • layout a una colonna, detto anche monolitico;
  • layout a due colonne;
  • layout a tre colonne;

Dal punto di vista tecnico, le diverse strategie per definire il layout delle pagineWeb possono essere così schematizzate:

  • layout fisso, in cui il layout mantiene le proprie dimensioni indipendentemente dalla dimensione della finestra del browser o del testo;
  • layout fluido o liquido, dove le dimensioni del layout cambiano in modo proporzionale alle dimensioni della finestra;
  • layout elastico, in cui le dimensioni del layout cambiano in modo proporzionale alla dimensione del testo;
  • layout ibrido, che combina aree fisse e aree le cui dimensioni variano dinamicamente.


Per fare ciò definiamo una serie di elementi:

  • header, che definisce un'intestazione;
  • nav, che definisce una serie di collegamenti di navigazione;
  • section, che definisce una sezione di un documento;
  • article, che definisce un contenuto autonomo;
  • aside, che definisce un contenuto oltre a quello autonomo;
  • footer, che definisce un piè di pagina.


Facciamo un esempio:


HTML

<section>
    <nav> Nav </nav>
    <article> Article </article>
    <aside> Aside </aside>
</section>
<footer> Footer </footer>


CSS

section{display: flex;}
nav {width: 20%; background-color: lightgray; color: black; height: 900px;}
article {width: 60%; background-color: grey; color: black; height: 900px;}
aside {width: 20%; background-color: darkgray; color: black; height: 900px;}
footer {bottom: 0; position: fixed; background-color: black; color: white; width: 100%; height: 100px;}
