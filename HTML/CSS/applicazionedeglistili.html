Applicazione degli stili


I fogli di stile consentono di definire lo stile di un particolare tag o di una famiglia di tag HTML attraverso una regola di stile. Una regola di stile è formata da una sequenza di coppie rappresentate da un nome separato dai due punti dal suo valore, separate tra di loro dal punto e virgola e racchiuse tra parentesi graffe.
Esempio:

p
{
    nome: valore;
    ......;
}


A differenza del linguaggio HTML, i fogli di stile CSS sono case sensitive, cioè distinguono tra caratteri minuscoli e maiuscoli: le regole di stile si scrivono tutte in minuscolo. In generale, il nome si riferisce a una proprietà di stile ed è un termine formato da una o due parole, in questo caso unite da trattino, mentre il valore è il dato a esso associato. Uno stile si applica a uno o più marcatori, chiamati selettori CSS, che devono essere digitati prima della regola di stile, senza indicarne però le parentesi angolari.
Gli stili possono essere utilizzati sostanzialmente secondo tre diverse modalità:

  • stili in linea;
  • stili a livello di documento;
  • stili esterni.
