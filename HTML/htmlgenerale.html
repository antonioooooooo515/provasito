Generale


L’HTML è un linguaggio per la scrittura di documenti ipertestuali che è stato studiato per essere utilizzato all’interno del WWW. È basato sia sul protocollo per la trasmissione di documenti (HTTP), che sulla notazione perl’indirizzamento delle risorse (URL). Le pagine di un sito (costituite da documenti HTML) sono memorizzate su un server Web e vengono visualizzate dal browser del computer client attraverso l’utilizzo di un browser Web, come per esempio Chrome, Firefox, Opera, Safari e Internet Explorer. Quando l’utente digita l’indirizzo Web (URL) della pagina che vuole visitare, il sistema operativo richiede, attraverso una connessione HTTP, la pagina al server in cui è memorizzata e la scarica affinché possa essere visualizzata dal browser. Quando il server non trova la pagina, per errore di digitazione oppure per altri motivi, restituisce il codice di errore 404.
