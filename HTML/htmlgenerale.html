Generale


L’HTML (HyperText Markup Language) è l'elemento costitutivo più basilare del Web. Definisce il significato e la struttura dei contenuti web. Il termine Hypertext indica che si hanno collegamenti tra le pagine Web tra loro, all'interno di un singolo sito Web o tra siti diversi. Caricando contenuti su Internet e collegandoli a pagine create da altre persone, diventi un partecipante attivo al World Wide Web. HTML utilizza il markup per annotare testo, immagini e altro contenuto da visualizzare in un browser Web. Esso include anche elementi come per esempio p, per la creazione di paragrafi. Un elemento HTML, in un documento, è separato dagli altri attraverso un tag, che consiste nel nome dell'elemento racchiuso da < e >. Il nome di un elemento all'interno di un tag non fa distinzione tra maiuscole e minuscole. Cioè, può essere scritto in maiuscolo, minuscolo o misto. I tre tag principali sono <html>, <head>, utilizzato per l'intestazione del file e <body>, utilizzato per il corpo del file.


Tag HTML

I tag sono normalmente in numero pari, per poter rappresentare l’inizio e la fine di un’istruzione. Prendono il nome di start-tag e end-tag (per esempio, <h1> e </h1>). Un end-tag ha la stessa forma dello start-tag eccetto per la barra (/) che deve precedere il testo tra le parentesi. Alcuni elementi possono includere un attributo o proprietà, ovvero un’informazione aggiuntiva contenuta all’interno dello start-tag.


Implementazione di una pagina HTML

Una pagina HTML può essere scritta secondo tre diverse modalità:

  • attraverso la scrittura manuale del codice con un editor di testo, come Notepad++;
  • attraverso la generazione automatica del codice con software di tipo WYSIWYG, come Dreamweaver;
  • ottenendo direttamente il sito completo grazie a CMS.
