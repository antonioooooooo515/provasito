Input


Il tag <input> è utilizzato per interagire con l'utente.
Sintassi:


<input type = "tipo">


Possiede, inoltre, diversi attributi. Eccoli:


Button

L'attributo button definisce un bottone. Esempio:


<input type = "button">





Checkbox

L'attributo checkbox definisce un insieme di opzioni limitate tra cui l'utente può scegliere. Esempio:


<input type = "checkbox" id = "esempio">
<label for = "esempio"> Primo </label>
<input type = "checkbox" id = "esempio">
<label for = "esempio"> Secondo </label>
<input type = "checkbox" id = "esempio">
<label for = "esempio"> Terzo </label>





Color

L'attributo color è utilizzato per l'inserimento di un colore attraverso una paletta. Esempio:


<input type = "color">





Date

L'attributo date è utilizzato per l'inserimento di una data. Esempio:


<input type = "date">





Datetime-local

L'attributo datetime-local è utilizzato per l'inserimento di data e ora. Esempio:


<input type = "datetime-local">





Email

L'attributo email è utilizzato per l'inserimento di un indirizzo email. Esempio:


<input type = "email">





File

L'attributo file è utilizzato per l'inserimento di un file attraverso un pulsante. Esempio:


<input type = "file">





Hidden

L'attributo hidden definisce un campo di input non visibile all'utente. Vine eutilizzato, di solito, per includere dati che non possono essere modificati o visualizzati quando viene inviato un modulo. (Anche se il valore non è visibile nella pagine, può essere modificato utilizzando gli strumenti di sviluppatore da "Visualizza sorgente" nei vari browser). Esempio:


<input type = "hidden">





Image

L'attributo image definisce un immagine come pulsante d'invio. (Il percorso è specificato dall'attributo src.) Esempio:


<input type = "image" src = "percoso/risorsa">





Month

L'attributo month definisce un campo dove l'utente può selezionare un mese e un anno. Esempio:


<input type = "month">





Number

L'attributo number definisce un campo di input numerico, che può anche accettare restrizioni su quali numeri vengono accettati. Esempio:


<input type = "number">





Password

L'attributo password definisce un campo password. I caratteri inseriti vengono visualizzati sottoforma di asterisco (*). Esempio:


<input type = "password">





Radio

L'attributo radio definisce un pulsante di opzione, che consente di selezionare SOLO UNA tra quelle disponibili. Esempio:




<input type = "radio" id = "primo" value = "primo" name = "scelta">
<label for = "primo"> Primo </label>
<input type = "radio" id = "secondo" value = "secondo" name = "scelta">
<label for = "secondo"> Secondo </label>
<input type = "radio" id = "terzo" value = "terzo" name = "scelta">
<label for = "terzo"> Terzo </label>





Range

L'attributo range definisce un controllo per inserire un valore compreso tra altri due (per esempio tra 0 e 10). Le restrizioni vengono effettuare secondo i seguenti attributi:

  • max, che indica il valore massimo;
  • min, che indica il valore minimo;
  • step, che indica gli intervalli di numeri possibili;
  • value, che indica un valore predefinito;


<input type = "range">





Reset

L'attributo reset definisce un pulsante di ripristino dei valori nel modulo a quelli predefiniti.


<input type = "reset">





Search

L'attributo search è utilizzato per i campi di ricerca (si comporta come un normale campo di testo).


<input type = "search">





Submit

L'attributo submit definisce un pulsante che invia tutti i valori del modulo ad un gestore, specificato nell'attributo action.


<input type = "submit">





Tel

L'attributo tel è utilizzato per i campi che devono contenere un numero di telefono.


<input type = "tel">





Text

L'attributo text definisce un campo di input di testo a singola riga.


<input type = "text">





Time

L'attributo time definisce un campo per inserire un'oram, senza alcun fuso orario.


<input type = "time">





Url

L'attributo url è utilizzato per i campi di input contenenti indirizzi URL.


<input type = "url">





Week

L'attributo week definisce un campo per selezionare una settimana e un anno.


<input type = "week">
