Input


Il tag <input> è utilizzato per interagire con l'utente attraverso i suoi attributi. Eccoli:

Button

L'attributo "button" definisce un bottone.

Checkbox

L'attributo "checkbox" definisce un insieme di opzioni limitate tra cui l'utente può scegliere.

Color

L'attributo "color" è utilizzato per l'inserimento di un colore attraverso una paletta.

Date

L'attributo "date" è utilizzato per l'inserimento di una data.

Datetime-local

L'attributo "datetime-local" è utilizzato per l'inserimento di data e ora.

Email

L'attributo "email" è utilizzato per l'inserimento di un indirizzo email.

File

L'attributo "file" è utilizzato per l'inserimento di un file attraverso un pulsante.

Hidden

L'attributo "hidden" definisce un campo di input non visibile all'utente. Vine eutilizzato, di solito, per includere dati che non possono essere modificati o visualizzati quando viene inviato un modulo. (Anche se il valore non è visibile nella pagine, può essere modificato utilizzando gli strumenti di sviluppatore da "Visualizza sorgente" nei vari browser.)

Image

L'attributo "image" definisce un immagine come pulsante d'invio. (Il percorso è specificato dall'attributo "src".)

Month

L'attributo "month" definisce un campo dove l'utente può selezionare un mese e un anno.

Number

L'attributo "number" definisce un campo di input numerico, che può anche accettare restrizioni su quali numeri vengono accettati.

Password

L'attributo "password" definisce un campo password. I caratteri inseriti vengono visualizzati sottoforma di asterisco (*).

Radio

L'attributo "radio" definisce un pulsante di opzione, che consente di selezionare SOLO UNA tra quelle disponibili.



Range

L'attributo "range" definisce un controllo per inserire un valore compreso tra altri due (per esempio tra 0 e 10). Le restrizioni vengono effettuare secondo i seguenti attributi:

  • max, che indica il valore massimo;
  • min, che indica il valore minimo;
  • step, che indica gli intervalli di numeri possibili;
  • value, che indica un valore predefinito;

Reset

L'attributo "reset" definisce un pulsante di ripristino dei valori nel modulo a quelli predefiniti.

Search

L'attributo "search" è utilizzato per i campi di ricerca (si comporta come un normale campo di testo).

Submit

L'attributo "submit" definisce un pulsante che invia tutti i valori del modulo ad un gestore, specificato nell'attributo "action".

Tel

L'attributo "tel" è utilizzato per i campi che devono contenere un numero di telefono.

Text

L'attributo "text" definisce un campo di input di testo a singola riga.

Time

L'attributo "time" definisce un campo per inserire un'oram, senza alcun fuso orario.

Url

L'attributo "url" è utilizzato per i campi di input contenenti indirizzi URL.

Week

L'attributo "week" definisce un campo per selezionare una settimana e un anno.
