Liste numerate e puntate


Le liste o elenchi si possono dividere in tre categorie:

  • elenchi numerati o ordinati in modo progressivo;
  • elenchi puntati o non ordinati;
  • elenchi di definizione.

Per gli elenchi numerati si utilizza il tag <ol> (Ordered List, Lista Ordinata). Per quelli puntati si utilizza il tag <ul> (Unordered List, Lista non Ordinata). Ogni elemento della lista deve inoltre essere preceduto dal tag <li> (List Item).
Esempio:


<ul>

<li> Primo elemento </li>
<li> Secondo elemento </li>
<li> Terzo elemento </li>
<li> Quarto elemento </li>

</ul>

Gli elenchi di definizione, invece, sono costituiti da coppie di elementi composte da un termine:

  • inizio lista (<dl>), Definition List;
  • fine lista (<dt>), Definition Term;
  • definizione (<dd>), Definition Defined.
