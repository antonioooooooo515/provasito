Classi universali


Una classe universale si ottiene scrivendo il nome della classe come selettore e anteponendo a essa il punto. Esempio:



 .evidenziato
 {

color: white;
font-family: Arial, sans-serif;
background-color: darkmagent

 }

Per applicare una classe a un marcatore HTML dobbiamo scrivere il nome della classe da associare mediante l’attributo class=”nome classe” del tag. Il nome della classe deve essere scritto senza il punto:



 <p class= "evidenziato">
   Paragrafo evidenziato
 </p>




Il risultato sarà il seguente:


Paragrafo evidenziato.
