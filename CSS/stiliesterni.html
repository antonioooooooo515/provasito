Stili esterni


A differenza delle due modalità precedenti, utilizzate per applicare uno stile a un singolo documento, possiamo utilizzare lo stile esterno per applicare gli stessi stili a più documenti. Per ottenere un foglio di stile esterno è necessario salvare le proprietà di stile desiderate all’interno di un file con estensione "".css". Per dichiarare, all’interno del documento HTML, che vogliamo includere le regole presenti nel foglio di stile esterno, dobbiamo inserire nella <head> il marcatore <link rel>, nel modo seguente:


<link rel=”stylesheet” type=”text/css” href= “style.css”></link>


In questo caso, all’interno del file "style.css", sono contenute le regole di stile che verranno applicate al documento HTML. L’attributo "type" definisce il tipo di contenuto del file esterno, che per i fogli di stile sarà sempre “text/css”; attraverso l’attributo "media" possiamo invece fare riferimento al dispositivo in cui il file verrà visualizzato: "all" indica tutti i dispositivi di visualizzazione, "screen" è usato per il browser sullo schermo, "print" per la stampante, "handheld" per dispositivi mobili (tablet, smartphone). Possiamo anche specificare più elementi, separandoli tra di loro con la virgola.
