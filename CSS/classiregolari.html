Classi regolari


Le classi regolari permettono di definire più stili applicabili a uno stesso tipo di marcatore. Per definire una classe regolare è sufficiente scrivere il nome del marcatore seguito dal punto e quindi dalla classe:



 p.paragrafo1
 {

color: white;
font-family: Poppins;
background-color: darkmagenta;

 }

 p.paragrafo2
 {

color: white;
font-family: Poppins;
background-color: darkviolet;

 }

Si tratta di classi più rigide in quanto associabili solo al marcatore HTML per il quale sono state definite: se infatti vengono richiamate da un marcatore diverso non producono alcun effetto.



<p class= "paragrafo1"> Primo paragrafo.< /p>


<p class= "paragrafo2"> Secondo paragrafo.< /p>


Il risultato sarà il seguente:


Primo paragrafo.

Secondo paragrafo.
