Stili in cascata


Le regole di stile possono essere applicate sia in sequenza che in cascata (cascading). L’esempio seguente mostra come applicare le regole di stile in sequenza al selettore div:


div{ font-family: Times; font-size: 10px;}
div{ font-family: Arial; font-size: 12px;}


Lo stesso risultato può essere ottenuto applicando uno stile in cascata rispetto allo stesso selettore:


div
{
    font-family: Arial;
    font-size: 12px;
}
