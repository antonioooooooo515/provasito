Classi


Il termine "classe" deriva dalla programmazione a oggetti e indica un elemento che può essere istanziato varie volte. Inquesto caso, la classe CSS è un selettore di una regola di stile che può essere applicato a diversi marcatori.


Quando vengono definiti più stili applicabili a uno stesso tipo di marcatore o a qualsiasi marcatore, si utilizzano le classi CSS. Esistono diversi tipi di classi:

  • classi universali o indipendenti;
  • classi regolari;
  • pseudoclassi.
