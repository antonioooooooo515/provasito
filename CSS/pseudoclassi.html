Pseudoclassi


Mediante le pseudoclassi possiamo modificare lo stile di un elemento HTML, come per esempio quando selezioniamo una casella di testo, oppure quando il mouse passa sopra a un collegamento ipertestuale. Per dichiarare una pseudoclasse utilizziamo la sintassi seguente:


selettore: pseudoclasse { dichiarazioni; }


Pseudoclassi dinamiche


Le pseudoclassi dinamiche consentono di modificare le impostazioni che contraddistinguono lo stato di un marcatore, spesso in seguito all’azione di un utente.



:hover


Si applica al passaggio del mouse sopra a un collegamento ipertestuale. La sua applicazione più comune è quella che consente la modifica dell’aspetto di link e pulsanti al passaggio del mouse. Facciamo un esempio con questo link. Prova a far passare il mouse sopra il link qui sotto:



Hover


Per fare ciò:


.esempio:hover
{
    background-color: #E34672;
    padding: 20px 25px;
    font-size: 28px;
    color: white;
}






:active


Serve per modificare la formattazione del link quando e mentre esso viene attivato dall’utente. L’elemento è in stato active durante la pressione del pulsante del mouse su di esso, per tutto il tempo che intercorre tra il clic e il rilascio del pulsante. Clicca qui sotto:



Active


Per fare ciò:


.esempio:active
{
    background-color: #24252A;
    padding: 20px 25px;
    font-size: 28px;
    color: white;
}








:focus


Questa pseudoclasse viene attivata quando un elemento riceve il focus, quando cioè può accettare eventi attraverso la tastiera, per esempio l’inserimento di testo in un modulo.



Prova a digitare:




Per fare ciò:


.esempio:focus
{
    background-color: #24252A;
    color: white;
}
