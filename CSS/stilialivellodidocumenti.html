Stili a livello di documento


Gli stili a livello di documento stabiliscono lo stile dei tag HTML presenti nello stesso documento e si collocano all’interno della coppia di tag "<style>...</style>" posti nell’intestazione del documento.


Esempio:

<head>
  <style>
  {
    p
    {
      font-family: arial;
    }
  }
  </style>
</head>

<body>
  <p>
    Titolo
  </p>
</body>
