Vorresti centrare un div ma non sai come?
Imparalo ora!

Primo metodo

Text-align: center

Centrami!

.outer-box
{
  width: 200px;
  height: 200px;
}

p.inner-box
{
  text-align: center
  line-height: 200px;
}

Secondo metodo

Dispaly: table

Centrami!

.outer-box
{
  display: table;
}

p.inner-box
{
  display: table-cell
  vertical-align: middle;
  text-align: center;
}

Terzo metodo

Position: absolute v1

Centrami!
.outer-box
{
  position: relative;
}

div.inner-box
{
  postion: absolute
  top: 50%;
  left: 50%;
  transform: translate(-50%,-50%);
}

Quarto metodo

Position: absolute v2

Centrami!
.outer-box
{
  position: relative;
}

div.inner-box
{
  postion: absolute
  inset: 0;
  margin: auto;
  width: fit-content;
  height: fit-content;
}

Quinto metodo

Margin: 0 auto

Centrami!
.outer-box
{
  width: 200px;
  padding: 100px 0;
}

div.inner-box
{
  margin: 0 auto
  width: fit-content;
}

Sesto metodo

Display: flex

Centrami!
.outer-box
{
  display: flex;
  align-items: center;
  justify-content: center;
}
