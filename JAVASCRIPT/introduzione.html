Introduzione


Uno dei metodi JAVASCRIPT principali è getElementById(). Esso fa in modo che ogni elemento con un determinato ID si comporti in un modo specifico. Esempi:





Javascript può cambiare il contenuto di un paragrafo.




JavaScript può cambiare lo stile di un elemento HTML, come per esempio la sua dimensione.




JavaScript può mostrare elementi nascosti.

Hello JavaScript!





In HTML, il codice JavaScript viene inserito tra i tag <script> e </script>. Esempio:




  <script>
  {

document.getElementById("prova").innerHTML = "My first Javascript!";

  }
  </script>

È possibile inserire un numero qualsiasi di script in un documento HTML. Possono essere inseriti nel tag <head>, nel tag <body> o in entrambi.

JavaScript in <head>


  <head>
    <script>
    {

document.getElementById("prova").innerHTML = "My first Javascript!";

    }
    </script>
  </head>

JavaScript in <body>


  <body>
    <script>
    {

document.getElementById("prova").innerHTML = "My first Javascript!";

    }
    </script>
  </body>

JavaScript esterno


 <script src="myScript.js"> </script>
