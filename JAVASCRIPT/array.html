Array


Un array è una variabile speciale, che può contenere più di un valore al suo interno.
Esempio:



const computers = ["MSI", "HP", "Samsung"];


Perchè usarli?

In caso di un elenco di elementi, la memorizzazione di essi in singole variabili potrebbe essere simile a questa:



const computer1 = "MSI";
const computer2 = "HP";
const computer2 = "Samsung";


Tuttavia, la soluzione più semplice sarà sempre l'utilizzo di un array.


Creazione di un array

Esistono due metodi per creare un array:



PRIMO METODO

const array_name = ["item1", "item2", ...];

Esempio:
const computers = ["MSI", "HP", "Samsung"];



SECONDO METODO

const array_name = new Array ("item1", "item2", ...);

Esempio:
const computers = new Array ("MSI", "HP", "Samsung");





Accesso agli elementi di un array

Si può accedere a un elemento dell'array facendo riferimento al numero di indice. Esempio:



const computers = ["MSI", "HP", "Samsung"];
let first = computers[0];





Modifica di un elemento di un array

Questa istruzione cambia il valore del primo elemento di un array. Esempio:



const computers = ["MSI", "HP", "Samsung"];
computers[0] = "Huawei";





Accesso all'array completo

Con JavaScript, è possibile accedere all'intero array facendo riferimento al nome dell'array. Esempio:



const computers = ["MSI", "HP", "Samsung"];
document.getElementById ("arraycomputer") .innerHTML = computers;
