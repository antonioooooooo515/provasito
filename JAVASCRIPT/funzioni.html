Funzioni


Una funzione Javascript è un blocco di codice progettato per eseguire un attività particolare. Vengono eseguite quando "qualcosa" le invoca.
Esempio:



function myFunction(p1, p2) {

return p1 * p2;

}


Sintassi della funzione

Una funzione Javascript è definita con la parola chiave "function", seguita da un nome e dalle parentesi tonde. I nomi delle funzioni possono contenere lettere, cifre, trattini bassi e segni di dollaro. Le parentesi possono includere i nomi dei parametri separati da virgole: (parametro1, parametro2, ...). Il codice della funzione è racchiuso tra parentesi graffe.
Esempio:



function name(parametro1, parametro2, ...) {

//code to be executed

}


I parametri della funzione sono elencati tra parantesi tonde nella definizione della funzione. Gli argomenti della funzione sono i valori ricevuti dalla funzione quando viene richiamata. All'interno della funzione, gli argomenti (i parametri) si comportano come variabili locali.


Invocazione di una funzione

Il codice all'interno della funzione verrà eseguito quando "qualcosa" la invocherà:


  • quando si verifica un evento (quando un utente fa clic su un pulsante);
  • quando viene invocato (chiamato) dal codice JavaScript;
  • automaticamente (auto-invoco).

Funzione ritorno

Quando JavaScript raggiunge lo stato di return, la funzione interrompe l'esecuzione. Se la funzione è stata invocata da un'istruzione, JavaScript "ritornerà" per eseguire il codice dopo l'istruzione di richiamo. Le funzioni spesso calcolano un valore che deve essere "restituito" al "chiamante".
Esempio:



let x = myFunction(5, 7);

//La funzione è stata chiamata, il valore sarà ritornato con x


function myFunction(a, b) {

return a * b;

}

//La funzione ritorna il prodotto tra a e b. Il valore della variabile x sarà 35
