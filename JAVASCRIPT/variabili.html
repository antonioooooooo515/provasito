Variabili


Le variabili sono contenitori per la memorizzazione dei dati. Esempio:


  var x = 10;
  var y = 5;
  var z = x + y;

Variabili let

Le variabili definite con let non possono essere ridichiarate. Le variabili definite con let devono essere dichiarate prima dell'uso. Le variabili definite con let hanno Block Scope.
Per esempio, non è possibile fare questo:


  let x = 10;
  let x = 5;

 

// La variabile è gia stata dichiarata in precedenza, quindi il suo valore rimane 10



  let x = 10;   // Qui x è 10
 {
    let x = 5;   // Qui x è 5
 {  

// Qui x è 10

Variabili const

Le variabili definite con const non possono essere ridichiarate. Le variabili definite con const non possono essere riassegnate. Le variabili definite con const hanno Block Scope.
Per esempio, non è possibile fare questo:


  const PI;
  PI = 3.14;
 

// Questo darà un errore



  const x = 10;   // Qui x è 10
 {
    const x = 5;   // Qui x è 5
 {  

// Qui x è 10
