Dichiarazioni


Programmi JavaScript

Un programma è è un elenco di istruzioni che devono essere eseguite da computer. In un linguaggio di programmazione, queste istruzioni di programmazione sono chiamate istruzioni.



Dichiarazioni JavaScript

Le istruzioni JavaScript sono composte da: valori, operatori, espressioni, parole chiave e commenti. Ogni istruzione è separata dalle altre dal punto e virgola. Esempio:



document.getElementById("prova").innerHTML = "Hello world";





Parole chiave JavaScipt

Le istruzioni JavaScript spesso iniziano con una parole chiave per identificare l'azione da eseguire. Ecco un elenco:


Parole chiave Descrizione
var Dichiara una variabile
let Dichiara una variabile di blocco
const Dichiara una costante di blocco
if Contrassegna un blocco di istruzioni da eseguire su una condizione
switch Contrassegna un blocco di istruzioni da eseguire in casi diversi
for Contrassegna un blocco di istruzioni da eseguire in un ciclo
function Dichiara una funzione
return Esce da una funzione
try Implementa la gestione degli errori in un blocco di istruzioni

Le parole chiave non possono essere utilizzare come nomi per le variabili.
